21  * This file contains the definitions for the config switcher task.
22  * It controls all tasks assigned to virtual buttons.
23  * If a new configuration should be loaded from the storage, all
24  * previously loaded tasks are deleted and new tasks are loaded.
25  *
26  * A slot configuration is provided by the config_storage, which is
27  * controlled by this module. The config_switcher itself triggers the
28  * slot switch if a new slotname is posted to the incoming queue.
29  *
30  * The config switcher also provides a task which is used to trigger
31  * a slot switch on a virtual button action.
32  *
33  */
