uint8_t taskJoystickConfig::mode

Release mode.

  • 0 Axis/Slider/Button value is not released on VB release flag
  • 1 Value is set to idle position (512 for axis, 0 for sliders) or button is released on VB release flag
Note
If type is set to BUTTON_RELEASE, this value is ignored.
