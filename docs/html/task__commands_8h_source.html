25  * Supported AT-commands:
26  (sent via serial interface, 115200 baud, using spaces between parameters. Enter (<cr>, ASCII-code 0x0d) finishes a command)
27 
28  AT returns "OK"
29  AT ID returns identification string (e.g. "FLipMouse V2.0")
30  AT BM <uint> puts button into programming mode (e.g. "AT BM 2" -> next AT-command defines the new function for button 2)
31  virtual buttons are defined in common.h
32 
33  USB HID commands:
34 
35  AT CL click left mouse button
36  AT CR click right mouse button
37  AT CM click middle mouse button
38  AT CD click double with left mouse button
39 
40  AT PL press/hold the left mouse button
41  AT PR press/hold the right mouse button
42  AT PM press/hold the middle mouse button
43 
44  AT RL release the left mouse button
45  AT RR release the right mouse button
46  AT RM release the middle mouse button
47 
48  AT WU move mouse wheel up
49  AT WD move mouse wheel down
50  AT WS <uint> set mouse wheel stepsize (e.g. "AT WS 3" sets the wheel stepsize to 3 rows)
51 
52  AT MX <int> move mouse in x direction (e.g. "AT MX 4" moves cursor 4 pixels to the right)
53  AT MY <int> move mouse in y direction (e.g. "AT MY -10" moves cursor 10 pixels up)
54  AT RO <uint> rotate stick orientation (e.g. "AT RO 180" flips x any y movements)
55 
56  AT JX <int> set joystick x axis (e.g. "AT JX 512" sets the x-axis to middle position)
57  AT JY <int> set joystick y axis (e.g. "AT JY 1023" sets the y-axis to full up position)
58  AT JZ <int> set joystick z axis (e.g. "AT JZ 0" sets the z-axis to lowest position)
59  AT JT <int> set joystick z turn axis (e.g. "AT JR 512" sets the rotation to middle position)
60  AT JS <int> set joystick slider position (e.g. "AT JS 512" sets the slider to middle position)
61  AT JP <int> press joystick button (e.g. "AT JP 1" presses joystick button 1)
62  AT JR <int> release joystick button (e.g. "AT JP 2" releases joystick button 2)
63  AT JH <int> set joystick hat position (e.g. "AT JH 45" sets joystick hat to 45 degrees)
64  possible values are: 0, 45, 90, 135, 180, 225, 270, 315 and -1 to set center position)
65 
66  AT KW <string> keyboard write string (e.g." AT KW Hello!" writes "Hello!")
67  AT KP <string> key press: press/hold keys identifier
68  (e.g. "AT KP KEY_UP" presses the "Cursor-Up" key, "AT KP KEY_CTRL KEY_ALT KEY_DELETE" presses all three keys)
69  AT KH <string> key hold: sticky hold keys (key will be pressed until next button press or "AT KH" command)
70  (e.g. "AT KH KEY_A" presses the "A" key until "AT KH KEY_A" is called again)
71  for a list of supported key identifier strings see below !
72 
73  AT KR <string> key release: releases all keys identified in the string
74  AT RA release all: releases all currently pressed keys and buttons
75 
76  Housekeeping commands:
77 
78  AT SA <string> save settings and current button modes to next free eeprom slot under given name (e.g. AT SAVE mouse1)
79  AT LO <string> load button modes from eeprom slot (e.g. AT LOAD mouse1 -> loads profile named "mouse1")
80  AT LA load all slots (displays names and settings of all stored slots)
81  AT LI list all saved mode names
82  AT NE next mode will be loaded (wrap around after last slot)
83  AT DE delete EEPROM content (delete all stored slots)
