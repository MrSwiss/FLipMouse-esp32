Detailed Description

FUNCTIONAL TASK - Execute macros.

This module is used as functional task for sending macros to the command parser.

Macros in context of FLipMouse/FABI are a string of concatenated AT commands, equal to normal AT commands sent by the host.

Note
AT command separation is done by a semicolon (';')
