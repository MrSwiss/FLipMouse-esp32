Detailed Description

FUNCTIONAL TASK - Infrared remotes.

This module is used as functional task for sending pre-recorded infrared commands. Recording of an infrared command is done here, but NOT within a functional task.

Pinning and low-level interfacing for infrared is done in hal_io.c.
