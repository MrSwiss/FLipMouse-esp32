22  * This file contains the abstraction layer for storing FLipMouse/FABI
23  * slots to a storage system. In case of the ESP32 we use FAT
24  * with wear leveling.
25  * PLEASE ADJUST THE MAKEFILE TO USE 512Byte SECTORS AND MODE SAFETY!
26  * We don't do any safe copying, just building a checksum to detect
27  * faulty slots.
28  *
29  * This module can be used to store a config struct (generalConfig_t)
30  * and the corresponding virtual button config structs to the storage.
31  *
32  * Following commands are implemented here:
33  * -) list all slot names
34  * -) load default slot
35  * -) load next or previous slot
36  * -) load slot by name or number
37  * -) store a given slot
38  * -) delete one slot
39  * -) delete all slots
40  *
41  * This module starts one task on its own for maintaining the storage
42  * access.
43  * All methods called from this module will block via a semaphore until
44  * the requested operation is finished.
45  *
46  * @see generalConfig_t
47  */
